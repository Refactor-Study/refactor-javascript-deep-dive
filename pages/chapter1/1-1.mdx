자바스크립트 데이터 타입은 크게 두 가지로 나뉘어 집니다. **기본형과 참조형**인데요.<br/>
일반적으로 기본형은 할당이나 연산시 복제되고, 참조형은 참조된다고 알려져 있습니다.

![data_type](/images/chapter1/data_type.png)
[그림1 데이터 타입의 종류]

> 💡&nbsp;&nbsp;그러나 엄밀히 말하면 두 개 모두 복제가 이뤄집니다!

기본형은 **값이 담긴 주솟값을 바로 복제**하는 반면,<br/>
참조형은 **값이 담긴 주솟값들로 이루어진 묶음을 가르키는 주솟값을 복제**한다는 점이 다릅니다.


![data_type](/images/chapter1/mini_Immutability.png)

기본형은 **불변성(Immutability)**, 즉 값이나 상태를 변경할 수 없습니다.<br/>
a라는 변수를 선언하고, 다음 라인에 값을 5로 변경할 수 있는데 ... 불변성이라니 !<br/>
왜 이런 특성을 가지고 있다고 한걸까요?

```
let a = 10;
a = 5; // 에러가 발생하지 않습니다
```
